# AI Learn

AI Learn is a single-page, static educational site that introduces artificial intelligence concepts and learning paths. It provides:
- A guided overview of AI fundamentals, machine learning, and deep learning.
- Interactive sections with animations, filters, and visualizations.
- Clickable algorithm cards that open dedicated detail pages powered by local data.

## Disclaimer
This project was created to test the Opus 4.6 LLM. The data and content have not been reviewed.
